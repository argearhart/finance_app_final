Import Members (CSV)

Upload a CSV file to import member data.

Required CSV Format:

Your CSV file should have these exact column headers:

  • Business Name - The name of the business
  • Phone - Phone number
  • E-mail - Email address
  • zip_code_address - Street address
  • city - City name
  • Membership type - Membership category
  • Renewal Date - When membership expires (MM-DD-YYYY format)

Optional columns:

  • Join Date - When they joined (MM-DD-YYYY format)
  • Contact Person - Primary contact name
  • Status - active, inactive, etc.
  • Notes - Any additional notes
